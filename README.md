# personal website! 

This is the repository for my website. 
Inspired by other people's personal websites, I decided to make my own- making a website would be a great place to learn some frontend, and perhaps deployment or webhosting experience if I ever get a custom domain.

access it [here](https://arcueun.vercel.app)

If you ever wanted to run locally, do 
```bash
npm install # for packages

npm run dev # start local server
```
